This is a [React](https://reactjs.org) + [TypeScript](https://www.typescriptlang.org/) boilerplate built with [Vite](https://vitejs.dev).

## Getting started

1. Install dependencies.

   ```bash
   pnpm install
   ```

2. Start dev server with hot reload at http://localhost:5173/
   ```bash
   pnpm run dev
   ```

## Commands

- Run build 
  ```bash
  pnpm run build
  ```
- Run tests and watch
  ```bash
  pnpm run test
  ```
